Detailed Description

This task rotates the turntable which holds the rest of the assembly while a fire has not been detected by the thermal camera. The turntable is rotated with a high torque motor. When a fire is detected, this motor's rotation is halted while the fire is extinguished. When extinguished, the motor's rotation resumes.
